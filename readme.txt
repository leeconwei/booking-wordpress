Simple photography studio booking plugin with multi-hour selection.

== Description ==

Features:
- Visual time slot selection
- Multi-hour booking
- Prevent overlapping bookings
- WooCommerce payment support
- Admin booking calendar

== Installation ==

1. Upload plugin
2. Activate plugin
3. Add shortcode [studio_booking_form]
